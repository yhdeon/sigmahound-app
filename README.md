# Setup Guide
1. `git clone https://github.com/yhdeon/sigmahound-app`
2. `cd sigmahound-app`
3. `pip install -r  requirements.txt`
4. `export PERPLEXITY_API_KEY="<refer to user manual>"`
5. `python3 app.py`
